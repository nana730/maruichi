## 開発言語

- Ruby 3.3.0
* Rails 7.1.5

## 開発手順
```
$ git clone git@github.com:nana730/coffee_shop.git
$ cd coffeeshop
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
